# Svesome

fontawesome v6 icons wrapper in svelte;

## install
```bash
  npm i svesome
```
### props
 required props are to be correct 
 * type (fa-solid,fa-brands) etc
 * icon (fa-home) etc
 * size (fa-2xl,fa-3xl) etc

## usage
```js
  import Svesome from "svesome";

<Svesome type="fa-solid" icon="fa-house" size="fa-2xl" />

```
